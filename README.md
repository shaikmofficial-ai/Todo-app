# 📝 Todo Application

A simple and user-friendly Todo Application built using **Spring Boot**, **Thymeleaf**, and **Bootstrap**. This project allows users to manage daily tasks efficiently through a clean web interface.

## 🚀 Features

* Add new tasks
* View all tasks
* Mark tasks as completed/incomplete
* Toggle task status
* Delete tasks
* Responsive Bootstrap-based UI
* Server-side rendering with Thymeleaf
* Persistent task storage using Spring Data JPA

## 🛠️ Tech Stack

### Backend

* Java
* Spring Boot
* Spring Data JPA

### Frontend

* Thymeleaf
* Bootstrap 5
* HTML5
* CSS3

### Database

* H2 Database / MySQL (depending on configuration)

### Build Tool

* Maven

## 📷 Application Preview

The application provides a clean dashboard where users can:

## ⚙️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/shaikmofficial-ai/Todo-app.git
```

### Navigate to the Project

```bash
cd Todo-app
```

### Run the Application

```bash
mvn spring-boot:run
```

or run:

```text
TodoApplication.java
```

from your IDE.

## 🌐 Access the Application

After starting the application:

```text
http://localhost:8080
```
